## ⚙️ How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd django-backend-portfolio
   ```

2. **Activate the environment & install dependencies:**
   ```bash
   # Activate your portfolio environment
   portfolio_env\Scripts\activate
   pip install pillow django
   ```

3. **Run database migrations:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

4. **Start the Django local development server:**
   ```bash
   python manage.py runserver
   ```
   Open `http://127.0.0` in your browser.
